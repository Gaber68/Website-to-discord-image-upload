# Discord Image Uploader

Upload and send images to Discord via webhook.

## Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=YOUR_GITHUB_URL)

## How to Use

1. Get your Discord webhook URL from Server Settings → Integrations → Webhooks
2. Paste the webhook URL into the form
3. Upload an image
4. Add an optional message
5. Click "Send to Discord"
